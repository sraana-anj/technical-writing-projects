## **User Guide: Installing Python and Running Your First Script**

*Written by Raana sk.*

---

### **Introduction**

This guide helps beginners install Python 3.11 on Windows and run their first Python program. You don’t need any prior coding experience. The instructions are simple, visual, and beginner-friendly.

---

### **System Requirements**

Before you begin, make sure you have:

* A computer running **Windows 10** or newer

* **Internet access** to download the installer

* **Administrator rights** to install software

* At least **200 MB** of free disk space

---

### **Step 1: Go to the Official Python Website**

1. Open your web browser.

2. Visit the official Python download page: [https://www.python.org/downloads/](https://www.python.org/downloads/).

3. Click on the large yellow **Download Python 3.11** button.

---

### 

### **Step 2: Start the Installation**

1. Once the setup file downloads, open it.

2. Check the box that says **“Add Python to PATH.”**

3. Click **Install Now.**

4. Wait for the installation to complete — it usually takes a minute or two.

*(Insert screenshot of installer window highlighting “Add Python to PATH.”)*

---

### **Step 3: Verify the Installation**

1. Open the **Command Prompt** (search “cmd” in the Start menu).

Type this command:

 `python --version`

You should see something like:

 `Python 3.11.0`

If you get that message, your installation worked\!

---

### **Step 4: Run Your First Python Program**

1. Open **IDLE** (Python’s built-in editor). You can find it in your Start menu.

In the editor, type this line:

 `print("Hello, World!")`

2. Click **Run → Run Module** or press **F5.**  
   The output window should display:

    `Hello, World!`
